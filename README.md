# WP Template Coder

WP Template Coder 是一个强大的 Android Studio 插件，用于管理、共享和同步代码模板。它提供了完整的模板管理解决方案，支持团队协作和版本控制。

## 功能特点

### 本地模板管理
- 快速访问模板设置
- 导入/导出模板文件
- 支持文件模板和实时模板
- 选择性导出并预览模板
- 导出后快速分享功能

### 模板共享功能
- 通过远程服务器与团队成员共享模板
- 基于角色的访问控制
- 简单的服务器配置
- Git 版本控制支持
- 按创建时间排序的模板列表

### 用户界面
- 直观的工具窗口
- 文件模板和实时模板分类管理
- 操作反馈清晰
- 友好的错误提示

## 安装要求

- Android Studio 2025.1.1 或更高版本
- JDK 21 或更高版本
- Kotlin 2.1.0 或更高版本

## 安装步骤

1. 在 Android Studio 中打开 Settings/Preferences
2. 选择 Plugins > Install Plugin from Disk
3. 选择下载的插件文件 (.zip)
4. 重启 Android Studio

## 基本使用

### 打开插件窗口
1. 点击顶部菜单 `View > Tool Windows > WP Coder`
2. 或使用快捷键打开工具窗口（默认位于右侧）

### 主要功能区
- 顶部：工具栏，包含主要操作按钮
- 中部：模板列表显示区域
- 底部：状态和提示信息

### 基本操作
1. **查看模板**
   - 切换标签页查看不同类型模板
   - 使用搜索框快速查找模板

2. **创建模板**
   - 点击 "+" 按钮新建模板
   - 选择模板类型（文件模板/实时模板）
   - 编辑模板内容

3. **编辑模板**
   - 双击模板项目进行编辑
   - 修改模板内容和变量
   - 保存更改

## 模板管理

### 导出模板
1. 点击工具栏的导出按钮
2. 选择要导出的模板
3. 选择导出位置
4. 确认导出

### 导入模板
1. 点击工具栏的导入按钮
2. 选择模板文件
3. 确认导入
4. 检查导入结果

### 支持的模板类型

#### 文件模板
- Activity 模板
- Fragment 模板
- Layout XML 模板
- 自定义 View 模板
- 数据类模板
- 接口类模板
- 工具类模板

#### 实时模板（Live Templates）
- 常用方法模板
- 初始化代码模板
- 网络请求模板
- 数据库操作模板
- 权限处理模板

## 模板共享

### 配置服务器
1. 点击设置按钮
2. 输入服务器地址
3. 配置 API 密钥
4. 测试连接

### 上传模板
1. 选择要分享的模板
2. 点击上传按钮
3. 添加描述信息
4. 确认上传

### 下载模板
1. 浏览远程模板列表
2. 选择需要的模板
3. 点击下载按钮
4. 确认集成到本地

## 权限说明

### 用户角色
1. **管理员（Admin）**
   - 完全访问权限
   - 可以上传、下载、删除模板
   - 管理用户权限

2. **编写者（Writer）**
   - 可以上传和下载模板
   - 不能删除模板

3. **读者（Reader）**
   - 只能下载和使用模板
   - 不能修改或上传
